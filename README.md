# LOL好友清理工具

基于 LCU API 的英雄联盟慢速删除好友工具，采用 Electron + Vue 3 构建，拥有现代化的 UI 界面。

## 功能特点

- 自动连接英雄联盟客户端
- 显示完整好友列表（包含在线状态）
- 多种删除模式：
  - 删除选中的好友
  - 删除全部好友
  - 只删除离线好友
  - 删除全部（优先在线）
- 可调节删除速度（0.5秒 - 10秒）
- 实时显示删除进度

## 安装

```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 打包 Windows 版本
npm run build:win
```

## 使用方法

1. 启动英雄联盟客户端并登录
2. 运行本工具
3. 在好友列表中选择要删除的好友
4. 设置删除模式和速度
5. 点击删除按钮开始

## 速度选项

| 速度 | 间隔 | 说明 |
|------|------|------|
| 极速 | 0.5秒/个 | 风险较高 |
| 快速 | 1秒/个 | 有一定风险 |
| 正常 | 2秒/个 | 推荐 |
| 慢速 | 3秒/个 | 安全 |
| 超慢 | 5秒/个 | 非常安全 |
| 龟速 | 10秒/个 | 绝对安全 |

## 注意事项

- 需要英雄联盟客户端已启动并登录
- Windows 系统可能需要管理员权限运行
- 速度越快风险越高，建议选择正常或更慢的速度
- 删除操作不可撤销，请谨慎操作
